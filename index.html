¿De qué trata el curso?

El curso de Introducción a los Sistemas de Cómputo proporciona una base fundamental en el conocimiento del hardware y software de una computadora, el funcionamiento de los sistemas operativos y el entendimiento de las redes de computadoras. Se abordan conceptos esenciales para comprender cómo interactúan los distintos componentes que permiten el procesamiento y comunicación de datos.

Resumen de los Temas

1. Tipos de Red

Se explican las redes LAN, WAN, MAN, así como topologías, dispositivos de red y medidas de seguridad.

2. Sistema Operativo

Describe las funciones, arquitectura, componentes y tipos de sistemas operativos, además de virtualización y software libre.

3. Hardware y Software

Se estudian los componentes físicos (hardware), dispositivos periféricos, unidades de almacenamiento y el software de sistema y aplicación.
