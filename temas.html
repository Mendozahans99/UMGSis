  • Hub LAN
  • Firewall
  • Punto de acceso inalámbrico
Medios de conexión

  • Medios LAN
  • Medios inalámbricos

Cable cruzado

Un cable cruzado de Ethernet permite conexiones directas entre dispositivos similares (ej. PC a PC) mediante una distribución específica de los hilos (568A/568B).

Elementos para conexión segura

  • Conectores RJ-45: Estándar para cables de red con colores en orden específico
  • Hub: Repetidor que recibe y retransmite tramas de Ethernet
  • Switch: Interconecta segmentos de red a mayor velocidad con procesamiento de información
  • Comunicación full duplex: Bidireccional simultánea

Tipos de redes

Tipo Descripción Ejemplo
LAN Red de área local (geografía pequeña) Red doméstica o de oficina
WAN Red de área extensa (geografía amplia) Internet, redes corporativas
WLAN LAN inalámbrica WiFi
MAN Red de área metropolitana Red de una ciudad
SAN Red de área de almacenamiento Almacenamiento en red para servidores
PAN Red de área personal (hasta 10m) Conexión Bluetooth

Topologías de red

  • Punto a punto: Conexión directa entre dos nodos
  • Bus: Todos los nodos conectados a un cable principal
  • Estrella: Dispositivo central conectado a otros
  • Anillo: Nodos conectados circularmente
  • Malla completa: Todos los dispositivos conectados entre sí
  • Malla parcial: Algunas conexiones directas entre dispositivos
  • Híbrido: Combinación de topologías

Seguridad en redes

Principales amenazas:

  1. Robo de información
  2. Robo de identidad
  3. Pérdida o manipulación de datos
  4. Interrupción del servicio

Medidas de seguridad física:

  • Control de acceso físico a equipos
  • Protección contra amenazas ambientales (temperatura, humedad)
  • Protección eléctrica (picos de voltaje)
  • Mantenimiento adecuado

2. Sistema Operativo

Definición y funciones

Un sistema operativo (SO) es el programa que gestiona los procesos básicos de un sistema informático y permite la ejecución de otras operaciones. Funciones principales:

  • Planificación de procesos
  • Gestión de dispositivos
  • Seguridad
  • Control de procesos
  • Gestión de memoria
  • Gestión de archivos

Componentes principales

Componente Función
Gestión de procesos Administra la interacción de procesos con recursos
Gestión de memoria Controla uso de memoria principal
Sistema de archivos Gestiona archivos y directorios (FAT32, NTFS, etc.)
Sistema de protección Brinda niveles de seguridad
Controladores Programas que permiten comunicación con hardware

Características clave

  • Multiusuario: Soporta múltiples usuarios
  • Multitarea: Ejecuta múltiples programas simultáneamente
  • Plug and Play: Detección automática de hardware

Tipos de sistemas operativos

Windows

Creado por Microsoft, popular en entornos domésticos y empresariales.

Linux

Sistema de código abierto, basado en Unix, altamente personalizable.

macOS

Sistema de Apple, basado en Unix, conocido por su interfaz gráfica.

Interfaces de usuario

  • GUI (Graphical User Interface): Interfaz gráfica con ventanas, iconos
  • CLI (Command Line Interface): Interfaz de línea de comandos

Software Libre vs. Comercial

Aspecto Software Libre Software Comercial
Costo Sin costo de licencia Requiere pago de licencia
Distribución Puede redistribuirse libremente No se puede redistribuir
Seguridad Menos vulnerable a virus Más propenso a virus
Actualizaciones Comunidad global contribuye Solo el desarrollador oficial

3. Hardware y Software

Modelo de von Neumann

Arquitectura básica de computadoras con 5 componentes:

  1. Unidad de entrada
  2. Unidad de salida
  3. Unidad de memoria
  4. Unidad de control
  5. Unidad aritmética y lógica

Características: uso de sistema binario, programas almacenados en memoria, ejecución secuencial.

Componentes del CPU

  • Unidad de Control: Decodifica instrucciones
  • ALU: Realiza operaciones aritméticas y lógicas
  • Registros: Almacenan datos temporales
  • Contador de programa: Sigue la secuencia de ejecución

Unidades de medida

Unidad Símbolo Equivalencia
Kilobyte KB 1024 bytes
Megabyte MB 1024 KB
Gigabyte GB 1024 MB
